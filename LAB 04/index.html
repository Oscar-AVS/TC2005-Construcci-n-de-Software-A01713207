¿Qué diferencias y semejanzas hay entre Java y JavaScript?

  

    Para empezar la única semejanza que comparten además del nombre que claramente puede ser similar, es que ambos pueden ser usados para programar orientado a objetos, sin embargo no hay más.  En cuanto a diferencias, encontramos que la primer diferencia es el tipado Java es un lenguaje de programación altamente tipado, o en otras palabras, impone restricciones estrictas sobre los tipos de datos mientras que JavaScript es un lenguaje de tipado dinámico,  la otra gran diferencia radica en cómo se ejecuta cada uno, Java por su parte debe ejecutarse en Java Virtual Machine (JVM),  de esta manera puede correrse el código de manera independiente mientra que  JavaScript está integrado en HTML  y requiere un navegador.

¿Cómo se declara una variable con alcance local dentro de una función?

  

    Para declarar o crear una variable con alcance local en un función se utilizan palabras reservadas tales como var, let y const.
  


  
¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

  

    Es considerado cómo una mala práctica, ya que reduce la legibilidad provocando acoplamiento y efectos secundarios impredecibles. Esto dificulta el mantenimiento y puede introducir errores de seguridad difíciles de detectar.
